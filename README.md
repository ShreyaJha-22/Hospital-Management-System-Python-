# Hospital Management System

A Python-based desktop terminal application built to streamline healthcare administrative operations, including managing records for patients, doctors, nurses, and staff.

## Features
- **Authentication:** Secure user signup and sign-in verification steps.
- **Administrative Control:** Full CRUD operational support (Create, Read, Update, Delete) to handle doctor, nurse, and support staff logs.
- **Patient Management:** Automated pathways for patient admissions, database tracking, and structured billing discharge verifications.
- **Relational Storage:** Relational schemas mapping datasets directly via custom structured SQL execution statements.

## Tech Stack
- **Language:** Python 3.x
- **Database:** MySQL
- **Connector:** `mysql-connector-python`
